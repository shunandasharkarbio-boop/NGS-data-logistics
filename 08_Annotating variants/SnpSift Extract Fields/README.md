## Variant Annotation Fields

The annotated VCF generated by LoFreq and SnpEff contains the following key fields used for downstream variant interpretation.

| Field | Description |
|-------|-------------|
| CHROM | Chromosome or reference sequence where the variant is located. |
| POS | Genomic position of the variant. |
| REF | Reference allele. |
| ALT | Alternative allele. |
| QUAL | Variant quality score assigned by LoFreq. |
| DP | Total sequencing read depth. |
| AF | Alternative allele frequency. |
| SB | Strand bias P-value from Fisher's Exact Test. |
| DP4 | Forward/Reverse reference and alternative allele read counts. |
| ANN[*].EFFECT | Predicted functional effect of the variant. |
| ANN[*].IMPACT | Predicted impact severity (HIGH, MODERATE, LOW, MODIFIER). |
| ANN[*].GENE | Gene containing the variant. |
| ANN[*].AA_POS | Affected amino acid position. |
| ANN[*].HGVS_C | HGVS coding DNA notation. |
| ANN[*].HGVS_P | HGVS protein notation. |
